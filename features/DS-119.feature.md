# DS-119 – Dashboard displaying the right components

**Jira:** [DS-119 – Dashboard displaying the right components](https://legionqaschool.atlassian.net/browse/DS-119)  
**User story:** As an admin user, I want to see the correct dashboard.

Feature: Dashboard displaying the right components
  An authenticated admin lands on the Dashboard and sees the expected navigation
  blocks. Each block card routes to the corresponding application page.

  # Happy paths

  Scenario: Navigate to the Dashboard
    Given I am logged in as admin
    When I navigate to the Dashboard page
    Then I see the Dashboard with the right blocks: Programs, Calendar, Validation, AI Assist

  Scenario: Successfully navigate to Program Page
    Given I am on Dashboard
    When I click on Programs card
    Then I navigate to the Programs page

  Scenario: Successfully navigate to Calendar Page
    Given I am on Dashboard
    When I click on Calendar card
    Then I navigate to the Calendar page

  Scenario: Successfully navigate to Validation Page
    Given I am on Dashboard
    When I click on Validation card
    Then I navigate to the Validation page

  Scenario: Successfully navigate to AI Assist Page
    Given I am on Dashboard
    When I click on AI Assist card
    Then I navigate to the AI Assist page

  # Negative

  Scenario: Dashboard does not show unrelated product blocks
    Given I am logged in as admin
    When I navigate to the Dashboard page
    Then I do not see a block card named "Export"
    And I do not see a block card named "Scheduler"

  Scenario: Guest user cannot access Dashboard blocks
    Given I am not logged in
    When I attempt to open the Dashboard page directly
    Then I am redirected to the login page
    And I do not see the Programs block card

  # Edge cases

  Scenario: Dashboard blocks remain visible after browser reload
    Given I am logged in as admin
    And I am on the Dashboard page
    When I reload the Dashboard page
    Then I see the Dashboard with the right blocks: Programs, Calendar, Validation, AI Assist

  Scenario: Returning to Dashboard from Programs via sidebar
    Given I am logged in as admin
    And I am on the Programs page
    When I click the Dashboard navigation button
    Then I see the Dashboard with the right blocks: Programs, Calendar, Validation, AI Assist

  Scenario: Each dashboard card is keyboard-focusable and activatable
    Given I am on Dashboard
    When I tab to the Programs card and press Enter
    Then I navigate to the Programs page
